# Claude Statusline

## Version 1.1.0 Changes

### New Features
- **OAuth API Support**: Introduced OAuth authentication to enhance security and manage user sessions.
- **Improved Data Pipeline**: Optimized the data processing workflow for better performance and reliability.
- **Three-Tier Data Source Priority**: Implemented a three-tier approach to data sourcing, enhancing the system's ability to prioritize data retrieval based on predefined criteria.

## Troubleshooting
If you encounter issues using this version, consider the following:
- Ensure that your OAuth credentials are correct and have the necessary access rights.
- Monitor the logs for any errors related to the data pipeline.
- Review the data source configurations for potential misconfigurations.

For further assistance, please consult the support documentation or reach out to the support team.
